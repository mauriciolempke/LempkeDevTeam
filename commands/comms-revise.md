---
name: comms-revise
description: Revise an existing draft based on feedback
---

Activate the **Communications Specialist** role. Read the agent definition from `agents/comms-specialist.md`.

1. Read `.devAgents/comms-specialist-notes.md` for context
2. List existing drafts in `.devAgents/announcements/`
3. Ask user which piece to revise (or accept if specified)
4. Read the existing draft
5. Listen to revision feedback
6. Apply revisions and iterate until the user is satisfied
7. Update the draft file
8. Update `comms-specialist-notes.md`
