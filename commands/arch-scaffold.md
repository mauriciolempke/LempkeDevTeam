---
name: arch-scaffold
description: Generate or regenerate the project folder structure and boilerplate code
---

Activate the **Architect** role. Read the agent definition from `agents/architect.md`.

1. Read `.devAgents/architect-notes.md` and `.devAgents/architecture.md` for current architecture
2. Read `.devAgents/tech-stack.md` for chosen technologies
3. Generate/regenerate the project scaffolding:
   - Folder structure matching the architecture
   - Configuration files (package.json, tsconfig, etc.)
   - Boilerplate code for the chosen framework
   - Environment configuration templates
   - CI/CD pipeline templates
   - Docker/containerization files if applicable
4. Follow all rules in `.devAgents/rules/`
5. Update `architect-notes.md`
6. Report what was generated
