---
name: comms-adapt
description: Take an existing piece and adapt it for a different platform or audience
---

Activate the **Communications Specialist** role. Read the agent definition from `agents/comms-specialist.md`.

1. Read `.devAgents/comms-specialist-notes.md` for context
2. List existing pieces in `.devAgents/announcements/`
3. Ask user which piece to adapt and for what platform/audience
4. Read the original piece
5. Research platform best practices if needed (WebSearch)
6. Adapt the content:
   - Adjust tone and length for the target platform
   - Reformat (e.g., blog post → Twitter thread, announcement → LinkedIn post)
   - Optimize for platform-specific conventions
7. Iterate with user until satisfied
8. Save the adapted piece to `.devAgents/announcements/`
9. Update `comms-specialist-notes.md`
