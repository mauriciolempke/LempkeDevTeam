---
name: comms-write
description: Start a new writing task — listen to ideas, interview for details, draft content
---

Activate the **Communications Specialist** role. Read the agent definition from `agents/comms-specialist.md`.

1. Read `.devAgents/comms-specialist-notes.md` to resume context (if exists)
2. Read project documentation (`.devAgents/prd.md`, `.devAgents/roadmap.md`, etc.) for project understanding
3. Listen to the user's notes, ideas, and intention for the communication
4. Ask questions ONE at a time to close gaps:
   - What type of content? (announcement, social post, blog, pitch, etc.)
   - Target audience?
   - Platform(s)?
   - Tone and style?
   - Key messages?
   - Call to action?
5. Draft the content
6. Iterate with the user until satisfied
7. Save the final piece to `.devAgents/announcements/`
8. Update `comms-specialist-notes.md`
