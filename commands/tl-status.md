name: tl-status
description: Show task board status — running, done, blocked tasks and milestones
---

Activate the **Tech Lead** role. Read the agent definition from `agents/tech-lead.md`.

1. Read `.devAgents/tech-lead-notes.md` and `.devAgents/tasks/task-board.md`
2. Display:
   - Milestone overview (completed / in-progress / remaining)
   - Task summary per milestone (TODO / IN-PROGRESS / DONE / FAILED / BLOCKED)
   - Currently running subagents (if any)
   - Any blocked items and why
3. Check `.devAgents/changes/` for unprocessed CRs
4. Suggest next action
