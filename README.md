# LempkeDevTeam
Description: A suite of 12 specialized Claude Code agents for full-cycle application development

## What are the Agents?
The code within this repository consists of 12 innovative Claude Code agents designed specifically for end-to-end application development processes. Each agent is specialized for handling distinct responsibilities such as testing, deployment, debugging, monitoring, and more.

## How to Install the Agents on Claude Code

Follow the steps below to get started:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/mauriciolempke/LempkeDevTeam.git
   ```

2. **Navigate to the Project Directory**:
   ```bash
   cd LempkeDevTeam
   ```

3. **Follow the Individual Setup Instructions**:
   Each agent resides in its dedicated folder, containing a README file with detailed installation and usage instructions.

4. **Install Necessary Dependencies**:
   Run the following command for each desired agent:
   ```bash
   pip install -r requirements.txt
   ```

5. **Run the Agent**:
   Execute the main script to initialize the agent:
   ```bash
   python main.py
   ```
